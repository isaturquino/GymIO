## 📌 Sobre o Projeto

O **GymIO – Sistema de Gestão Interna para Academias** é uma aplicação desenvolvida para automatizar e centralizar a administração de academias, permitindo o controle de alunos, planos, pagamentos, equipamentos e acessos.

O sistema tem como objetivo otimizar processos internos, reduzir erros operacionais e melhorar o controle financeiro e administrativo, além de fornecer indicadores estratégicos por meio de um dashboard.

O projeto está sendo desenvolvido de forma incremental ao longo dos bimestres.

---

## ⚙️ Principais Funcionalidades

- 🔐 Autenticação de usuários (Login)
- 👤 Gestão de alunos (CRUD completo)
- 📋 Planos e matrículas
- 💰 Gestão financeira
- 🚪 Controle de acesso (catraca)
- 🏋️ Gestão de equipamentos
- 📊 Relatórios
- 📈 Dashboard com indicadores
